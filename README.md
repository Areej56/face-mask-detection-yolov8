# Face Mask Detection using YOLOv8

This repository contains a complete end-to-end implementation of a face mask
detection system using YOLOv8.

The full workflow (data preprocessing, XML to YOLO conversion, training, and
inference) is documented in a Jupyter Notebook.

## Notebook
- `notebooks/face-mask-detection-yolov8.ipynb`
